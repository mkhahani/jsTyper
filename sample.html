• Typer
  • Typer is a pure JavaScript utility to create typing text effect over a list of items.
  • It would be useful for text advertising, quotes or breaking news.
  • Typer is a good replacement for obsolete HTML "marquee" too.
  • It also supports XHTML sildes like this.
  • There are cool options to set for example typing speed, cursor and so on.
  • If move your mouse cursor over the slider, it will pause!
