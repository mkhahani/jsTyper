• jsTyper
  • jsTyper is a pure JavaScript utility to create typing text effect over a list of items.
  • It would be useful for text advertising, announcement or headline news.
  • jsTyper is a good replacement for obsolete HTML "marquee".
  • It also supports XHTML sildes like this.
  • There are cool options to set the behaviour of the typer.
  • If move your mouse cursor over the slider, it will pause!
